## コマンド集

#### IISインストールとHTML作成
```powershell
# IISインストール
Install-WindowsFeature web-server

# HTMLファイル作成
New-Item -ItemType file -Path C:\inetpub\wwwroot\index.html -Value "<h1>$($env:computername)</h1>"
```
